# Deyyyyyyn
ソ連国歌のデェェェェェェェンの音程を自由に高くできるアプリ
## デモ
※音がメインのアプリです

https://github.com/user-attachments/assets/f5505bc6-5536-4f13-bcaf-74ca45f574e9

## 機能
- ソ連国歌のデェェェェェェンが流せる
- Hueと対応したスライダーによりピッチを高くできる
- Hueの変化に応じて左側のソ連国旗の色が変わる

## 使用技術
- TypeScript(5.8.3)
- React(18.2)
- ブラウザのAudio系API
